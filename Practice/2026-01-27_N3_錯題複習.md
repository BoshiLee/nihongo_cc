### Q1 ❌ たびに vs うちに

**題目：** この映画は見る（　2　）涙が出る。

1. たびに ✓
2. うちに
3. ついでに
4. ながら

> この{映画|えいが}は{見|み}る<b><u>たびに</u></b>{涙|なみだ}が{出|で}る。

**翻譯：** 這部電影每次看都會流淚。

**你選：** うちに
**正解：** たびに

**解析：**
- **たびに** = 每當…就（重複動作，每次都發生）
- **うちに** = 趁著…／在…過程中（狀態持續期間）
- 這題強調「每次看」都會哭 → たびに

### Q2 ✓ 〜たて

**題目：** 彼女は焼き（　2　）のパンが好きだ。

1. あげ
2. たて ✓
3. 上がり
4. 終わり

> {彼女|かのじょ}は{焼|や}き<b><u>たて</u></b>のパンが{好|す}きだ。

**翻譯：** 她喜歡剛烤好的麵包。

**解析：** 連用形＋たて = 剛做完的狀態。焼く → 焼き → 焼きたて

### Q5 ❌ さえ…ば vs こそ

**題目：** 薬（　3　）飲めば、すぐ治りますよ。

1. だけ
2. さえ ✓
3. こそ
4. しか

> {薬|くすり}<b><u>さえ</u></b>{飲|の}めば、すぐ{治|なお}りますよ。

**翻譯：** 只要吃藥，馬上就會好。

**你選：** こそ
**正解：** さえ

**解析：**
- **さえ…ば** = 只要…就（唯一條件）
- **こそ** = 強調，但不接條件形「〜ば」
- 句型：名詞＋さえ＋條件形

### Q7 ❌ 〜ようとしない vs 気がしない

**題目：** 何度注意しても、彼は遅刻を（　4　）。

1. やめるつもりはない
2. やめようとしない ✓
3. やめまい
4. やめる気がしない

> {何度|なんど}{注意|ちゅうい}しても、{彼|かれ}は{遅刻|ちこく}を<b><u>やめようとしない</u></b>。

**翻譯：** 不管提醒多少次，他就是不肯改掉遲到的習慣。

**你選：** やめる気がしない
**正解：** やめようとしない

**解析：**
- **〜ようとしない** = 第三人稱頑固地拒絕做某事
- **気がしない** = 沒有那種感覺（用於直覺、預感）
- 「何度注意しても」強調對方的抗拒 → ようとしない

---

### Q8 ❌ 〜まい vs 気はない

**題目：** あんな辛い経験は二度と（　4　）と思う。

1. しようとしない
2. するつもりはない
3. しまい ✓
4. する気はない

> あんな{辛|つら}い{経験|けいけん}は{二度|にど}と<b><u>しまい</u></b>と{思|おも}う。

**翻譯：** 我想那樣痛苦的經歷絕不要再有了。

**你選：** する気はない
**正解：** しまい

**解析：**
- **〜まい** = 否定意志（絕不再…）
- 「二度と」＋「〜まい」是固定搭配，表示強烈決心
- 「気はない」語氣較弱，不如「〜まい」強烈

## 重點整理

| 文法 | 意思 | 關鍵點 |
| ---- | ---- | ---- |
| たびに | 每當…就 | 重複動作、每次 |
| うちに | 趁著… | 狀態持續期間 |
| さえ…ば | 只要…就 | 唯一條件（接條件形） |
| こそ | 正是、才是 | 強調（不接條件形） |
| 〜ようとしない | 頑固不肯做 | 第三人稱抗拒 |
| 〜まい | 絕不再… | 搭配「二度と」表決心 |
| 気がしない | 沒有那種感覺 | 直覺、預感 |
