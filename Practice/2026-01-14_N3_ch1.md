# N3 文法練習 - 2026-01-14
**日期：2026-01-14**
**範圍：複合動詞複習 + 對照與對應（に対して／について）**
**成績：4/5**

## 相關文法
- [[〜合う]] - 互相做某動作
- [[〜上がる]] - 自然完成（自動詞）
- [[〜たて]] - 剛〜的（新鮮狀態）
- [[について]] - 關於（主題）
- [[に対して]] - 對於（對象）

---

## 練習題

### 第1題 ✓ → [[〜合う]]
この映画について、友達と（　　）。

1. 話した
2. **話し合った** ← 正解
3. 話された
4. 話し上げた

**我的答案：** 2

**正解句：**
> この 映画[えいが]について、友達[ともだち]と 話[はな]し合[あ]った。

**翻譯：** 關於這部電影，和朋友討論了。

**解析：** 「**と**」表示共同對象 → 搭配「〜合う」表示互相討論。
---

### 第2題 ✓ → [[〜上がる]]
料理が（　　）ので、テーブルに運んでください。

1. でき上げた
2. **でき上がった** ← 正解
3. できて上がった
4. 作り上がった

**我的答案：** 2

**正解句：**
> 料理[りょうり]が でき上[あ]がったので、テーブルに 運[はこ]んでください。

**翻譯：** 料理做好了，請端到桌上。

**解析：** 「料理**が**」→ 自動詞「でき上がる」。「でき上げる」這個形態不存在！
---

### 第3題 ✓ → [[について]]
日本の文化（　　）レポートを書いている。

1. に対して
2. **について** ← 正解
3. にとって
4. として

**我的答案：** 2

**正解句：**
> 日本[にほん]の 文化[ぶんか]について レポートを 書[か]いている。

**翻譯：** 正在寫關於日本文化的報告。

**解析：** 「について」用於說明、調查、討論的**主題**。「レポートを書く」的主題是「日本の文化」。
---

### 第4題 ❌ → [[に対して]] / [[について]]
このサービスの問題点（　　）、お客様からたくさんの意見をいただいた。

1. **について** ← 正解
2. に対して
3. にとって
4. として

**我的答案：** 2

**正解句：**
> この サービスの 問題点[もんだいてん]について、お客様[きゃくさま]から たくさんの 意見[いけん]を いただいた。

**翻譯：** 關於這個服務的問題點，收到了很多客人的意見。

**解析：**
- 「について」= 關於某**主題**收到意見
- 「に対して」= 對某**對象**（人/事物）表達態度或行動

這題是「關於問題點」這個主題收到意見，所以用「について」。

**比較：**
| 文法 | 用法 | 例句 |
|:---|:---|:---|
| について | 主題 | 日本**について**話す（談論關於日本的事） |
| に対して | 對象 | 先生**に対して**失礼だ（對老師很失禮） |
---

### 第5題 ✓ → [[〜たて]]
彼女は（　　）の新入社員で、まだ仕事に慣れていない。

1. 入社して
2. 入社たて
3. **入社したて** ← 正解
4. 入社しての

**我的答案：** 3

**正解句：**
> 彼女[かのじょ]は 入社[にゅうしゃ]したての 新入社員[しんにゅうしゃいん]で、まだ 仕事[しごと]に 慣[な]れていない。

**翻譯：** 她是剛入職的新員工，還不習慣工作。

**解析：** 「〜たて」接**連用形**！
- 入社する → 入社し（連用形）→ 入社したて ✓
- 入社する → 入社（語幹）→ 入社たて ✗

## 錯題筆記

### 「について」vs「に対して」

| 文法 | 用法 | 例句 |
|:---|:---|:---|
| について | 主題（關於…） | 日本の文化**について**調べる |
| に対して | 對象（對於…） | 先生**に対して**失礼なことを言う |

**判斷方法：**
- 如果是「關於某主題」進行調查、討論、意見 → **について**
- 如果是「對某對象」表達態度、行動、反應 → **に対して**
