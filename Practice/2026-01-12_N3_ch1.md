# N3 文法練習 - Ch1
**日期：2026-01-12**
**範圍：動詞連用形＋合う／上がる／上げる**
**成績：3/5**

## 相關文法
- [[〜合う]] - 互相做某動作
- [[〜上がる]] - 自然完成（自動詞）
- [[〜上げる]] - 主動完成（他動詞）

---

## 題目與批改

### 第1題 ❌ → [[〜合う]]
その問題について、部長と（　　）必要があります。

1. 話す
2. **話し合う** ← 正解
3. 話される
4. 話してあげる

**我的答案：1**

**正解句：**
> その 問題[もんだい]について、部長[ぶちょう]と 話[はな]し合[あ]う 必要[ひつよう]があります。

**翻譯：** 關於那個問題，有必要和部長討論。

**解析：** 「と」表示共同對象，搭配「〜合う」表示**互相討論**。「話す」只是單向說話。

---

### 第2題 ❌ → [[〜上げる]]
やっと論文を（　　）。本当に大変だった。

1. 書いた
2. 書き上がった
3. **書き上げた** ← 正解
4. 書き合った

**我的答案：2**

**正解句：**
> やっと 論文[ろんぶん]を 書[か]き上[あ]げた。本当[ほんとう]に 大変[たいへん]だった。

**翻譯：** 終於把論文寫完了。真的很辛苦。

**解析：** 關鍵是助詞「**を**」！
- 「を」→ 他動詞 →「書き**上げる**」（我把它寫完）
- 「が」→ 自動詞 →「書き**上がる**」（它自己完成了）

### 第3題 ✓ → [[〜上がる]]
ケーキが（　　）ので、みんなで食べましょう。

1. 焼き上げた
2. **焼き上がった** ← 正解
3. 焼いた
4. 焼き合った

**我的答案：2**

**正解句：**
> ケーキが 焼[や]き上[あ]がったので、みんなで 食[た]べましょう。

**翻譯：** 蛋糕烤好了，大家一起吃吧。

**解析：** 「が」→ 自動詞 →「焼き上がる」（蛋糕自然烤好了）

### 第4題 ✓ → [[〜合う]]
あの二人は学生時代からずっと（　　）。

1. 愛している
2. 愛された
3. **愛し合っている** ← 正解
4. 愛してあげている

**我的答案：3**

**正解句：**
> あの 二人[ふたり]は 学生[がくせい]時代[じだい]からずっと 愛[あい]し合[あ]っている。

**翻譯：** 那兩個人從學生時代就一直互相相愛。

**解析：** 「愛し合う」= 互相相愛，強調雙向感情。「愛している」只是單向的愛。

### 第5題 ✓ → [[〜合う]]
チームで働くときは、お互いに（　　）ことが大切だ。

1. 助ける
2. 助けられる
3. **助け合う** ← 正解
4. 助けてもらう

**我的答案：3**

**正解句：**
> チームで 働[はたら]くときは、お互[たが]いに 助[たす]け合[あ]うことが 大切[たいせつ]だ。

**翻譯：** 在團隊工作時，互相幫助是很重要的。

**解析：** 「お互いに」＋「助け合う」= 互相幫助，完美搭配！

## 重點整理

| 文法 | 意思 | 重點 |
|:---|:---|:---|
| [[〜合う]] | 互相～ | 常搭配「と」「お互いに」 |
| [[〜上がる]] | 自然完成（自動詞） | 主語用「**が**」 |
| [[〜上げる]] | 主動完成（他動詞） | 受詞用「**を**」 |
