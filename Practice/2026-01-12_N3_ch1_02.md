# N3 文法練習 - Ch1 (第二回)
**日期：2026-01-12**
**範圍：動詞連用形＋かえる／忘れる／でき上がる**
**成績：4/5**

## 相關文法
- [[〜かえる]] - 重新做、換〜
- [[〜忘れる]] - 忘記做某動作
- [[〜上がる]] - 自然完成（自動詞）

---

## 題目與批改

### 第1題 ✓ → [[〜かえる]]
東京駅で新幹線から在来線に（　　）ください。

1. 乗る
2. **乗りかえて** ← 正解
3. 乗りかえって
4. 乗って

**我的答案：2**

**正解句：**
> 東京駅[とうきょうえき]で 新幹線[しんかんせん]から 在来線[ざいらいせん]に 乗[の]りかえて ください。

**翻譯：** 請在東京站從新幹線換乘在來線。

**解析：** 「乗りかえる」（換乘）是下一段動詞，て形是「乗りかえて」。選項3的「乗りかえって」是錯誤形態。

---

### 第2題 ✓ → [[〜忘れる]]
今朝、薬を（　　）しまった。

1. 飲んで
2. **飲み忘れて** ← 正解
3. 飲みかえて
4. 飲み上げて

**我的答案：2**

**正解句：**
> 今朝[けさ]、薬[くすり]を 飲[の]み忘[わす]れて しまった。

**翻譯：** 今天早上不小心忘了吃藥。

**解析：** 「〜てしまった」表示遺憾或不小心做了某事。「飲み忘れる」= 忘記吃（藥）。

---

### 第3題 ✓ → [[〜上がる]]
注文したケーキが（　　）ので、取りに行きます。

1. でき上げた
2. **でき上がった** ← 正解
3. できた上がった
4. 作り上げた

**我的答案：2**

**正解句：**
> 注文[ちゅうもん]した ケーキが でき上[あ]がったので、取[と]りに 行[い]きます。

**翻譯：** 訂的蛋糕做好了，我去拿。

**解析：** 主語是「ケーキ**が**」→ 需要自動詞 →「でき上がる」。「でき上げる」這個形態不存在。

---

### 第4題 ✓ → [[〜かえる]]
この文章は間違いが多いから、（　　）ください。

1. 書いて
2. 書き忘れて
3. **書きかえて** ← 正解
4. 書き上げて

**我的答案：3**

**正解句：**
> この 文章[ぶんしょう]は 間違[まちが]いが 多[おお]いから、書[か]きかえて ください。

**翻譯：** 這篇文章錯誤很多，請重寫。

**解析：** 「書きかえる」= 改寫、重寫。「書き上げる」是「寫完」，語意不符。

---

### 第5題 ❌ → [[〜忘れる]]
出かけるとき、エアコンを（　　）。

1. **消し忘れた** ← 正解
2. 消しかえた
3. 消し上げた
4. 消し合った

**我的答案：3**

**正解句：**
> 出[で]かけるとき、エアコンを 消[け]し忘[わす]れた。

**翻譯：** 出門的時候，忘記關冷氣了。

**解析：**
- 「消し**忘れる**」= 忘記關
- 「消し**上げる**」這個形態**不存在**！
- 「〜上げる」通常用於「徹底完成」的語感（書き上げる、磨き上げる），但「消す」不能這樣搭配。

## 重點整理

| 文法 | 意思 | 重點 |
|:---|:---|:---|
| [[〜かえる]] | 重新～、換～ | 乗りかえる、書きかえる、入れかえる |
| [[〜忘れる]] | 忘記做～ | 飲み忘れる、消し忘れる、置き忘れる |
| [[〜上がる]] | 完成（自動詞） | 主語用「**が**」，無「でき上げる」形態 |

## 錯題筆記

### 〜忘れる 常見搭配
- 飲み忘れる（忘記吃藥）
- 消し忘れる（忘記關）
- 書き忘れる（忘記寫）
- 置き忘れる（忘記帶走）
- 言い忘れる（忘記說）
