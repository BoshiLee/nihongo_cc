### Q1 ✓ 〜ようとしない

**題目：** 何度説明しても、弟は宿題を（　2　）。

1. やるつもりはない
2. やろうとしない ✓
3. やるまい
4. やらせない

> {何度|なんど}{説明|せつめい}しても、{弟|おとうと}は{宿題|しゅくだい}を<b><u>やろうとしない</u></b>。

**翻譯：** 不管解釋多少次，弟弟就是不肯做作業。

**解析：** 第三人稱頑固地拒絕做某事 → 〜ようとしない。

### Q2 ❌ 自發動詞 vs 使役

**題目：** 娘の結婚式で、つい子供のころのことを思い出して（　1　）しまった。

1. 泣かせて
2. 泣いて ✓
3. 泣かれて
4. 泣けて

> {娘|むすめ}の{結婚式|けっこんしき}で、つい{子供|こども}のころのことを{思|おも}い{出|だ}して<b><u>{泣|な}いて</u></b>しまった。

**翻譯：** 在女兒的婚禮上，不禁想起了小時候的事情而哭了。

**你選：** 泣かせて（讓別人哭）
**正解：** 泣いて（自己哭）

**解析：**
- 這句的主語是「我」，是自己哭了，不是讓別人哭
- 「泣かせて」= 使役形，讓（別人）哭
- 「泣いて」= 單純動詞，自己哭

### Q3 ✓ 〜まい（否定推測）

**題目：** こんな遠い所まで、彼が一人で来（　3　）。

1. ないつもりだ
2. たくない
3. るまい ✓
4. ようとしない

> こんな{遠|とお}い{所|ところ}まで、{彼|かれ}が{一人|ひとり}で<b><u>{来|く}るまい</u></b>。

**翻譯：** 這麼遠的地方，他應該不會一個人來吧。

**解析：** 對第三者行為的否定推測 → 〜まい。

### Q4 ✓ 使役被動

**題目：** 彼女に会って、昔のことを（　3　）された。

1. 思い出し
2. 思い出して
3. 思い出させ ✓
4. 思い出さ

> {彼女|かのじょ}に{会|あ}って、{昔|むかし}のことを<b><u>{思|おも}い{出|だ}させ</u></b>された。

**翻譯：** 見到她之後，被勾起了過去的回憶。

**解析：** 思い出す → 思い出させる（使役）→ 思い出させられる（使役被動，被迫回想起）。

### Q5 ❌ 〜気はない vs つもりはない

**題目：** 私がどんなに頼んでも、あの人は手伝う（　2　）。

1. まい
2. つもりはない
3. 気がしない
4. 気はない ✓

> {私|わたし}がどんなに{頼|たの}んでも、あの{人|ひと}は{手伝|てつだ}う<b><u>{気|き}はない</u></b>。

**翻譯：** 不管我怎麼拜託，那個人都沒有幫忙的意願。

**你選：** つもりはない（沒有打算）
**正解：** 気はない（沒有意願）

**解析：**
- 「つもりはない」= 沒有計畫/打算（偏向具體計畫）
- 「気はない」= 沒有那個意願/心情（偏向主觀態度）
- 這題強調的是「不管怎麼拜託都不願意」，重點是意願而非計畫

### Q6 ❌ 〜まい（否定意志）

**題目：** 二度とあんな失敗は（　4　）と心に決めた。

1. しまい
2. するまい ✓
3. しないつもり
4. しようとしない

> {二度|にど}とあんな{失敗|しっぱい}は<b><u>するまい</u></b>と{心|こころ}に{決|き}めた。

**翻譯：** 我下定決心絕不再犯那樣的錯誤。

**你選：** しようとしない（不肯做）
**正解：** するまい（絕不再做）

**解析：**
- 「〜まい」有兩個用法：①否定推測 ②否定意志（絕不...）
- 「心に決めた」= 下定決心 → 搭配否定意志的「〜まい」
- 「しようとしない」是描述第三者不肯做某事，不適合表達自己的決心

### Q7 ❌ 情感使役

**題目：** このニュースを聞いたら、みんなを（　3　）だろう。

1. 驚く
2. 驚かせる ✓
3. 驚かされる
4. 驚けさせる

> この ニュースを{聞|き}いたら、みんなを<b><u>{驚|おどろ}かせる</u></b>だろう。

**翻譯：** 聽到這個消息的話，應該會讓大家嚇一跳吧。

**你選：** 驚かされる（被嚇到）
**正解：** 驚かせる（讓...嚇一跳）

**解析：**
- 主語是「このニュース」（這個消息）
- 消息讓大家驚訝 → 使役形「驚かせる」
- 「驚かされる」= 被動，意思是「被嚇到」，主語要是被嚇的人

## 重點整理

| 文法 | 意思 | 關鍵點 |
| ---- | ---- | ---- |
| 〜まい | ① 否定推測 ② 否定意志 | 「心に決めた」搭配否定意志 |
| 気はない | 沒有意願 | 強調主觀態度（不想做） |
| つもりはない | 沒有打算 | 強調具體計畫 |
| 情感使役 | 讓某人... | 主語是「引發情緒的事物」 |
| 自發 vs 使役 | 自己做 vs 讓別人做 | 確認主語是誰在行動 |
