## 全題解析

### Q1 ✓ うちに

> {暖|あたた}かい<b><u>うちに</u></b> {出|で}かけましょう。

**翻譯：** 趁暖和的時候出門吧。

**解析：**
- **うちに** = 趁著…（在某狀態持續期間）
- 「暖かい」是狀態，趁著這狀態還在 → 用 **うちに**

---

### Q2 ✓ たびに

> {彼|かれ}は {日本|にほん}に {来|く}る<b><u>たびに</u></b> お{土産|みやげ}を {買|か}ってくれる。

**翻譯：** 他每次來日本都會買伴手禮給我。

**解析：**
- **たびに** = 每當…就（重複動作）
- 「来る」是重複動作，每次來都買 → 用 **たびに**

---

### Q3 ✓ ば〜ほど

> {練習|れんしゅう}すれば<b><u>する</u></b><b><u>ほど</u></b> {上手|じょうず}になる。

**翻譯：** 越練習越熟練。

**解析：**
- **ば〜ほど** = 越…越…
- 固定句型：動詞ば形 + 動詞辭書形 + ほど

---

### Q4 ✓ ほど

> あの {映画|えいが}は {見|み}れば {見|み}る<b><u>ほど</u></b> {好|す}きになる。

**翻譯：** 那部電影越看越喜歡。

**解析：**
- 同 Q3，**ば〜ほど** = 越…越…

---

### Q5 ❌ ことにした vs ようになった

> {来月|らいげつ}から {毎週|まいしゅう}ジムに {行|い}く<b><u>ことにした</u></b>。

**翻譯：** 我決定從下個月開始每週去健身房。

**解析：**
- **ことにした** = 自己決定（意志決定）
- **ことになった** = 被決定（外部決定、安排）
- **ようになった** = 變得能夠 / 開始習慣性地…
- 「每週去健身房」是自己主動決定 → 用 **ことにした**

**判斷方法：**

|  文法  | 意思  |  主語的意志  |
|  ----  |  ----  |  ----  |
| ことにした | 決定… | ⭕ 有意志 |
| ことになった | 被決定… | ❌ 無意志 |
| ようになった | 變得… | — 能力/習慣變化 |

---

### Q6 ✓ こそ

> {失敗|しっぱい}した<b><u>からこそ</u></b> {学|まな}べることがある。

**翻譯：** 正因為失敗了才能學到東西。

**解析：**
- **からこそ** = 正因為…才（強調原因）
- 強調「失敗」這個原因的重要性

---

### Q7 ❌ さえ vs こそ

> {彼女|かのじょ}は {挨拶|あいさつ}<b><u>さえ</u></b> しない {失礼|しつれい}な {人|ひと}だ。

**翻譯：** 她是連打招呼都不做的失禮的人。

**解析：**
- **さえ…ない** = 連…都不（舉極端例子表示程度）
- 「挨拶」是最基本的禮貌，連這個都不做 → 強調失禮程度
- **こそ** 是強調，但不用於「連…都不」的句型

**判斷方法：**

| 句型 | 意思 | 例句 |
| ---- | ---- | ---- |
| さえ…ない | 連…都不 | 挨拶**さえ**しない |
| さえ…ば | 只要…就 | 健康**さえ**あれば |
| こそ | 正是、才是 | 今度**こそ**成功したい |

---

### Q8 ✓ さえ…ば

> {健康|けんこう}<b><u>さえ</u></b> あれば {何|なん}でもできる。

**翻譯：** 只要有健康就什麼都能做。

**解析：**
- **さえ…ば** = 只要…就（唯一條件）
- 健康是唯一需要的條件 → 用 **さえ…ば**

## 重點整理

| 文法     | 意思    | 關鍵      |
| ---- | ---- | ---- |
| うちに    | 趁著…   | 狀態持續期間  |
| たびに    | 每當…就  | 重複動作    |
| ば〜ほど   | 越…越…  | 比較變化    |
| ことにした  | 自己決定  | 有意志     |
| ことになった | 被決定   | 無意志     |
| ようになった | 變得…   | 能力/習慣變化 |
| こそ     | 正是、才是 | 強調      |
| さえ…ない  | 連…都不  | 極端例子    |
| さえ…ば   | 只要…就  | 唯一條件    |
