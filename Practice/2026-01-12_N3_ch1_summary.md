## 錯題整理

### 錯題1｜第一回 第1題
**考點：〜合う（互相）**

> その 問題[もんだい]について、部長[ぶちょう]と（　　）必要[ひつよう]があります。

| 我的答案 | 正解 |
|:---:|:---:|
| 1. 話す | 2. **話し合う** |

**翻譯：** 關於那個問題，有必要和部長討論。

**錯誤原因：**
- 「話す」= 單向說話
- 「話し**合う**」= 互相討論、商量
- 關鍵提示：「**と**」（和～一起）→ 暗示雙向動作 → 用「〜合う」

**記憶口訣：** 看到「〜と」就想到「〜合う」！

### 錯題2｜第一回 第2題
**考點：〜上がる vs 〜上げる（自他動詞）**

> やっと 論文[ろんぶん]を（　　）。本当[ほんとう]に 大変[たいへん]だった。

| 我的答案 | 正解 |
|:---:|:---:|
| 2. 書き上がった | 3. **書き上げた** |

**翻譯：** 終於把論文寫完了。真的很辛苦。

**錯誤原因：**
- 關鍵是助詞「**を**」！
- 「論文**を**」→ 他動詞 →「書き**上げる**」（我把它寫完）
- 「論文**が**」→ 自動詞 →「書き**上がる**」（它自己完成了）

**記憶口訣：**
```
を → 上げる（他動詞：我完成它）
が → 上がる（自動詞：它完成了）
```

### 錯題3｜第二回 第5題
**考點：〜忘れる vs 〜上げる**

> 出[で]かけるとき、エアコンを（　　）。

| 我的答案 | 正解 |
|:---:|:---:|
| 3. 消し上げた | 1. **消し忘れた** |

**翻譯：** 出門的時候，忘記關冷氣了。

**錯誤原因：**
- 「消し**上げる**」這個形態**不存在**！
- 「〜上げる」只能接特定動詞（書く、磨く、焼く、揚げる...）
- 「消す」只能接「〜忘れる」

**「〜上げる」可搭配的動詞：**
| 可以 | 不可以 |
|:---|:---|
| 書き上げる | ~~消し上げる~~ |
| 磨き上げる | ~~開け上げる~~ |
| 焼き上げる | ~~閉め上げる~~ |
| 仕上げる | ~~見上げる~~（意思不同） |

### 錯題4｜第三回 第4題
**考點：〜たて（剛～的）的接續**

> 日本[にほん]へ（　　）ころは、日本語[にほんご]が 全然[ぜんぜん]わからなかった。

| 我的答案 | 正解 |
|:---:|:---:|
| 3. 来ての | 1. **来たての** |

**翻譯：** 剛來日本的時候，日語完全不懂。

**錯誤原因：**
- 「〜たて」接的是**動詞連用形**，不是て形！
- 「来**て**の」→ 錯誤形態
- 「来**たて**の」→ 正確（来る的連用形「来（き）」＋たて）

**「〜たて」的正確接續：**
```
動詞連用形 ＋ たて

焼く  → 焼き  → 焼きたて（剛烤好的）
来る  → 来（き）→ 来たて（剛來的）
揚げる → 揚げ  → 揚げたて（剛炸好的）
```

**記憶口訣：** 「たて」不是「て形」！是「連用形＋たて」！

## 重點複習清單

### 1. 助詞決定自他動詞
```
が → 自動詞（上がる、でき上がる）
を → 他動詞（上げる、書き上げる）
```

### 2.「〜合う」的提示詞
```
と（和～）  → 話し合う、助け合う
お互いに   → 助け合う、愛し合う
```

### 3. 複合動詞的搭配限制
不是所有動詞都能隨意組合！
- ~~消し上げる~~ → 消し忘れる ✓
- ~~来ての~~ → 来たての ✓

### 4.「〜たて」的接續
```
動詞連用形 ＋ たて（不是て形！）
```
