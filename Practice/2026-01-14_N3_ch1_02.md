## 練習題

### 第1題 ❌ → [[に対して]]
お客様からのクレーム（　　）、すぐに対応しなければならない。

1. について
2. **に対して** ← 正解
3. によって
4. にとって

**我的答案：** 1

**正解句：**
> お客様[きゃくさま]からの クレームに対[たい]して、すぐに 対応[たいおう]しなければならない。

**翻譯：** 對於客人的投訴，必須立即應對。

**解析：**
- 「対応する」（應對）是對某**對象**採取行動 → 用「**に対して**」
- 「について」是關於某主題「討論/調查」，這裡是「應對」不是「討論」
---

### 第2題 ❌ → [[によって]]
国（　　）習慣が違う。

1. に対して
2. について
3. **によって** ← 正解
4. として

**我的答案：** 1

**正解句：**
> 国[くに]によって 習慣[しゅうかん]が 違[ちが]う。

**翻譯：** 依國家不同，習慣也不同。

**解析：**
- 「によって」表示**判斷基準**（因…而異）
- 「国によって違う」= 依國家不同而不同（因人而異、因地而異的用法）
- 「に対して」是對某對象表達態度，不是「差異」的意思
---

### 第3題 ❌ → [[にとって]]
私（　　）、家族が一番大切だ。

1. として
2. に対して
3. **にとって** ← 正解
4. について

**我的答案：** 2

**正解句：**
> 私[わたし]にとって、家族[かぞく]が 一番[いちばん] 大切[たいせつ]だ。

**翻譯：** 對我而言，家人是最重要的。

**解析：**
- 「にとって」表示**主觀立場**（對…而言）
- 「私にとって大切」= 對我而言很重要（主觀看法）
- 「に対して」是對某對象表達態度/行動，不是表達「看法」
---

### 第4題 ❌ → [[として]]
彼は医者（　　）この病院で働いている。

1. にとって
2. **として** ← 正解
3. に対して
4. によって

**我的答案：** 1

**正解句：**
> 彼[かれ]は 医者[いしゃ]として この 病院[びょういん]で 働[はたら]いている。

**翻譯：** 他作為醫生在這家醫院工作。

**解析：**
- 「として」表示**身分、角色、資格**
- 「医者として働く」= 以醫生的身分工作
- 「にとって」是主觀立場（對…而言），不是身分

### 第5題 ✓ → [[に対して]]
先生（　　）失礼な態度を取ってはいけない。

1. について
2. として
3. にとって
4. **に対して** ← 正解

**我的答案：** 4

**正解句：**
> 先生[せんせい]に対[たい]して 失礼[しつれい]な 態度[たいど]を 取[と]ってはいけない。

**翻譯：** 不可以對老師採取失禮的態度。

**解析：**
- 「に対して」表示對某**對象**的態度/行動
- 「先生に対して失礼」= 對老師失禮（對象是老師）

## 重點整理

這幾個文法很容易混淆！記住以下區別：

| 文法 | 意思 | 關鍵詞 | 例句 |
|:---|:---|:---|:---|
| [[について]] | 關於（主題） | 討論、調查、說明 | 日本**について**調べる |
| [[に対して]] | 對於（對象） | 態度、行動、反應 | 先生**に対して**失礼だ |
| [[によって]] | 依據（基準） | 因…而異、手段 | 国**によって**違う |
| [[として]] | 作為（身分） | 身分、角色、資格 | 医者**として**働く |
| [[にとって]] | 對…而言（立場） | 主觀看法、價值觀 | 私**にとって**大切だ |

## 錯題筆記

### 判斷方法

1. **討論/調查某主題** → について
2. **對某對象表達態度/行動** → に対して
3. **因…而異** → によって
4. **以某身分做某事** → として
5. **對某人而言（主觀看法）** → にとって

### 容易混淆的組合

| 情境 | 正確 | 錯誤 |
|:---|:---|:---|
| 對客訴「應對」 | クレーム**に対して**対応する | ~~について~~ |
| 依國家「不同」 | 国**によって**違う | ~~に対して~~ |
| 對我「而言」重要 | 私**にとって**大切 | ~~に対して~~ |
| 以醫生「身分」工作 | 医者**として**働く | ~~にとって~~ |
