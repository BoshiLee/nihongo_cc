### 第6題 ✗
**你的答案：** 2. 建て上がった
**正解：** 4. 建ち上がった

> やっと 家[いえ]が 建[た]ち上[あ]がった。来月[らいげつ] 引[ひ]っ越[こ]しできる。

**翻譯：** 房子終於蓋好了。下個月可以搬家了。

**解析：**
- **關鍵：自動詞 vs 他動詞**
- 「が」→ 自動詞「建つ」→ 建ち上がる ✓
- 「を」→ 他動詞「建てる」→ 建て上げる
- 這題是「家**が**」所以要用自動詞「建つ」

| 助詞 | 動詞類型 | 用法 |
|:---|:---|:---|
| が | 自動詞 | 建つ → 建ち上がる |
| を | 他動詞 | 建てる → 建て上げる |

### 第8題 ✗
**你的答案：** 4. 卒業たて
**正解：** 2. 卒業したて

> 彼[かれ]は 大学[だいがく]を 卒業[そつぎょう]したてで、まだ 社会経験[しゃかいけいけん]が 少[すく]ない。

**翻譯：** 他剛大學畢業，社會經驗還很少。

**解析：**
- **「〜たて」接續：連用形 + たて**
- 卒業する → 卒業し（連用形）→ 卒業したて ✓
- 卒業する → 卒業（語幹）→ 卒業たて ✗

**記憶點：** 「〜たて」不是直接接語幹，要接連用形！

### 第9題 ✗
**你的答案：** 4. 合った
**正解：** 1. 合って

> 友達[ともだち]と 目[め]が 合[あ]って、二人[ふたり]とも 笑[わら]ってしまった。

**翻譯：** 和朋友眼神對上了，兩個人都笑了。

**解析：**
- 這個句子用て形連接兩個動作：「對上眼」→「笑了」
- 需要用て形「合って」來連接後面的「笑ってしまった」
- 如果用過去式「合った」，句子會在這裡結束，無法連接後面的動作

**句型：** 動詞て形 + 動詞（表示動作順序）

### ❌ 第8題：〜たて的接續

```
正確：連用形 + たて
卒業する → 卒業し → 卒業したて ✓
焼く → 焼き → 焼きたて ✓

錯誤：語幹 + たて
卒業する → 卒業 → 卒業たて ✗
```

---

### ❌ 第9題：て形的連接功能

```
動詞て形 + 動詞 = 表示動作順序

目が合って、笑った
（對上眼後，笑了）

如果用過去式「合った」，就無法連接後面的動作了。
```

## 重點整理

| 文法 | 關鍵規則 |
|:---|:---|
| 〜上がる/上げる | 「が」→上がる、「を」→上げる |
| 〜たて | 連用形＋たて（不是語幹！） |
| て形連接 | 表示動作順序或因果關係 |
| 自他動詞 | 看助詞「が/を」判斷 |
