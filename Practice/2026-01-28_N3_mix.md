## 批改結果

### 【第1題】✓ 〜合う

> {二人|ふたり}で<b><u>{考|かんが}え{合|あ}って</u></b>、やっと{答|こた}えが{見|み}つかった。

**翻譯：** 兩人一起討論思考，終於找到了答案。

**解析：**
- **〜合う** = 互相做某動作（需要雙方參與）
- 「二人で」暗示互動，所以用「考え合う」
- 「考えて」= 單純思考，不強調互動
- 「考えさせて」= 讓人思考（使役）
- 「考えられて」= 被思考（被動）

---

### 【第2題】✓ 〜たて

> このケーキは<b><u>{焼|や}きたて</u></b>だから、{早|はや}く{食|た}べてね。

**翻譯：** 這個蛋糕剛烤好，快點吃喔。

**解析：**
- **〜たて** = 剛剛做完（接動詞連用形）
- 「焼く」→ 連用形「焼き」→「焼きたて」
- ❌「焼いたて」→ て形不能接「たて」
- ❌「焼けたて」→「焼ける」是自動詞，蛋糕是被烤的
- ❌「焼くたて」→ 基本形不能接「たて」

---

### 【第3題】❌ 〜ようとしない

**你的答案：** 3. 変えまい
**正解：** 1. 変えようとしない

> {彼女|かのじょ}は{何|なに}を{言|い}っても、{自分|じぶん}の{意見|いけん}を<b><u>{変|か}えようとしない</u></b>。

**翻譯：** 不管說什麼，她就是不肯改變自己的意見。

**解析：**
- **〜ようとしない** = 第三人稱頑固地拒絕做某事（描述他人的抗拒態度）
- **〜まい** = 說話者自己的否定意志（「我絕不…」）
- 這題主語是「彼女」（第三人稱），描述她的抗拒行為 → 用「〜ようとしない」
- 「〜まい」用於第一人稱或推測，不適合描述他人持續的抗拒

### 【第4題】✓ 〜によって

> この{問題|もんだい}は{人|ひと}<b><u>によって</u></b>{答|こた}えが{違|ちが}う。

**翻譯：** 這個問題因人而異，答案不同。

**解析：**
- **〜によって** = 根據…、因…而異
- 「人によって違う」= 因人而異（固定搭配）
- 「について」= 關於…（主題）
- 「に対して」= 對於…（態度、對象）
- 「に関して」= 有關…（較正式的「關於」）

### 【第5題】❌ うちに

**你的答案：** 1. たびに
**正解：** 3. うちに

> {忙|いそが}しい<b><u>うちに</u></b>{旅行|りょこう}に{行|い}っておきたい。

**翻譯：** 想趁忙碌之前（還不忙的時候）先去旅行。

**解析：**
- **うちに** = 趁著…（狀態持續期間，強調要在某狀態結束前行動）
- **たびに** = 每當…就（重複動作）
- 這題的邏輯：「忙しいうちに」→ 趁著現在還忙（還有空檔）的時候
- ⚠️ 注意：這裡「忙しい」是反話，意思是「趁還沒變得太忙」
- 「たびに」需要重複動作，但旅行不是每次發生的事

**補充：〜ておく**
- **{行|い}っておきたい** = 想事先去（為將來做準備）
- 「〜ておく」= 事先做好某事、為之後做準備
- 整句：趁還不忙的時候，想**先**去旅行（之後可能沒機會了）

### 【第6題】❌ 迷惑受身

**你的答案：** 3. 踏ませて
**正解：** 2. 踏まれて

> {電車|でんしゃ}の{中|なか}で{足|あし}を<b><u>{踏|ふ}まれて</u></b>、{本当|ほんとう}に{迷惑|めいわく}だった。

**翻譯：** 在電車裡被踩到腳，真的很困擾。

**解析：**
- **迷惑受身**（被動表受害）= 被別人做了某事而感到困擾
- 「足を踏まれて」= 腳被踩了（受害者視角）
- ❌「踏ませて」= 讓人踩（使役）→ 邏輯不通，沒人會讓別人踩自己
- ❌「踏まされて」= 被迫踩（使役被動）→ 意思是「被迫去踩別人」

**補充：踏む的動詞變化**

| 形式    | 變化  | 意思     |     |
| ----- | --- | ------ | --- |
| 基本形   | {踏  | ふ}む    | 踩   |
| 被動形   | {踏  | ふ}まれる  | 被踩  |
| 使役形   | {踏  | ふ}ませる  | 讓人踩 |
| 使役被動形 | {踏  | ふ}まされる | 被迫踩 |

**迷惑受身句型：**
- 「（身體部位）を＋被動形」= 被（做了某事）而困擾
- 例：{足|あし}を{踏|ふ}まれた（腳被踩了）
- 例：{頭|あたま}を{叩|たた}かれた（頭被打了）
- 例：{肩|かた}を{押|お}された（肩膀被推了）

## 重點整理

| 文法      | 意思     | 關鍵判斷          |
| ------- | ------ | ------------- |
| 〜合う     | 互相…    | 雙方參與的動作       |
| 〜たて     | 剛剛做完   | 接動詞連用形        |
| 〜ようとしない | 頑固不肯做  | 第三人稱抗拒        |
| 〜まい     | 絕不再…   | 第一人稱意志        |
| 〜によって   | 因…而異   | 「人によって違う」固定搭配 |
| うちに     | 趁著…    | 狀態持續期間        |
| 迷惑受身    | 被…（受害） | 受害者視角 + 被動    |
