## 問題與批改

**第1題 ❌**
> 留学生の彼女は日本に（　　）ころ、電車の乗り方がわからなくて困った。

1. 来ての
2. 来たばかり ← 你的答案
3. 来たての ✓
4. 来るの

> {留学生|りゅうがくせい}の {彼女|かのじょ}は {日本|にほん}に <b><u> {来|き}たての </u></b> ころ、{電車|でんしゃ}の {乗|の}り {方|かた}が わからなくて {困|こま}った。

**翻譯：** 作為留學生的她，剛來日本的時候，不知道怎麼坐電車很困擾。

**解析：**
- **〜たて** = 剛剛做完某事（強調「新鮮」的狀態）
- **〜たばかり** = 剛剛做完某事（強調時間上「剛剛」）
- 關鍵：後面接「ころ」時，需要用「〜たての」（連體形）
- 「来たばかり」不能直接修飾「ころ」，要說「来たばかりの」

---

**第2題 ❌**
> 新しい政策（　　）、市民から反対の意見が多く出ている。

1. について ← 你的答案
2. に対して ✓
3. によって
4. にとって

> {新|あたら}しい {政策|せいさく}<b><u>に {対|たい}して</u></b>、{市民|しみん}から {反対|はんたい}の {意見|いけん}が {多|おお}く {出|で}ている。

**翻譯：** 對於新政策，市民提出了很多反對意見。

**解析：**
- **反対** 是對某對象的**態度/立場** → 用「に対して」
- 「について」用於「說明/討論主題」

**記憶口訣：**
- 賛成/反対 → **に対して**
- 説明/調査 → **について**

**第3題 ✓**
> 彼は何度注意されても、自分のやり方を（　　）。

1. 変えるつもりはない
2. 変えようとしない ✓
3. 変えまい
4. 変えないことにした

> {彼|かれ}は {何度|なんど} {注意|ちゅうい}されても、{自分|じぶん}の やり {方|かた}を <b><u>{変|か}えようと しない</u></b>。

**翻譯：** 他不管被提醒多少次，都不肯改變自己的做法。

**解析：** 「何度〜ても」+ 頑固抗拒 → **〜ようとしない**

---

**第4題 ❌**
> この仕事は経験者（　　）簡単だが、初心者（　　）難しい。

1. にとって / にとって ✓
2. として / として ← 你的答案
3. に対して / に対して
4. について / について

> この {仕事|しごと}は {経験者|けいけんしゃ}<b><u>にとって</u></b> {簡単|かんたん}だが、{初心者|しょしんしゃ}<b><u>にとって</u></b> {難|むずか}しい。

**翻譯：** 這份工作對有經驗的人來說簡單，但對初學者來說很難。

**解析：**
- 後面是**形容詞評價**（簡単/難しい） → **にとって**
- **として** = 作為（身分），後面接**動作**（働く、参加する）

**第5題 ❌**
> 子供のころは野菜が嫌いだったが、大人になって食べられる（　　）。

1. ことにした
2. ことになった
3. ようになった ✓
4. ようにした ← 你的答案

> {子供|こども}の ころは {野菜|やさい}が {嫌|きら}いだったが、{大人|おとな}に なって {食|た}べられる<b><u>ように なった</u></b>。

**翻譯：** 小時候討厭蔬菜，但長大後變得能吃了。

**解析：**

| 文法 | 意思 | 用法 |
| ---- | ---- | ---- |
| ようになった | 變得能夠… | 能力/習慣的**自然變化** |
| ようにした | 努力使… | **刻意努力**去做 |

- 這題強調「長大後自然變得能吃」→ **ようになった**
- 如果是「努力嘗試吃蔬菜」才用「ようにした」

**第6題 ✓**
> 彼女に会う（　　）、もっと好きになってしまう。

1. うちに
2. たびに ✓
3. ところに
4. ばかりに

> {彼女|かのじょ}に {会|あ}う<b><u>たびに</u></b>、もっと {好|す}きに なって しまう。

**翻譯：** 每次見到她，就更加喜歡了。

**解析：** 「每次」做某事 → **たびに**

**第7題 ✓**
> 会議で意見が分かれたので、もう一度みんなで（　　）ことにした。

1. 話す
2. 話し合う ✓
3. 話し上げる
4. 話しかける

> {会議|かいぎ}で {意見|いけん}が {分|わ}かれたので、もう {一度|いちど} みんなで <b><u>{話|はな}し {合|あ}う</u></b> ことに した。

**翻譯：** 因為會議上意見分歧，所以決定大家再討論一次。

**解析：** 「みんなで」= 大家一起（互相）→ **〜合う**

**第8題 ❌**
> 自分の失敗で周りの人を（　　）しまって、本当に申し訳ない。

1. 困って
2. 困らせて ✓
3. 困られて ← 你的答案
4. 困らされて

> {自分|じぶん}の {失敗|しっぱい}で {周|まわ}りの {人|ひと}を <b><u>{困|こま}らせて</u></b> しまって、{本当|ほんとう}に {申|もう}し {訳|わけ}ない。

**翻譯：** 因為自己的失敗讓周圍的人困擾了，真的很抱歉。

**解析：**
- **困らせる**（使役）= 讓別人困擾（我造成的）
- **困られる**（被動）= 被困擾（日語中「困られる」較少用）

**情感使役記憶：**

| 原形 | 使役形 | 意思 |
| ---- | ---- | ---- |
| 困る | 困らせる | 讓...困擾 |
| 悲しむ | 悲しませる | 讓...傷心 |
| 喜ぶ | 喜ばせる | 讓...高興 |

## 重點整理

| 弱點文法 | 正確用法 | 記憶點 |
| ---- | ---- | ---- |
| 〜たて vs 〜たばかり | 〜たての + 名詞 | 修飾名詞時用「たての」 |
| について vs に対して | 賛成/反対 → に対して | 態度用「に対して」 |
| にとって vs として | 評價 → にとって | 形容詞評價用「にとって」 |
| ようになった vs ようにした | 自然變化 → ようになった | 非刻意 = なった |
| 困らせる vs 困られる | 讓人困擾 → 困らせる | 使役形表達「讓別人...」 |
