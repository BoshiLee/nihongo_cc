## 題目與批改

### 第1題 ✓
（　　）パンはとてもおいしいです。

1. 焼いた
2. **焼きたての** ← 正解
3. 焼くの
4. 焼いている

**我的答案：2**

**正解句：**
> 焼[や]きたての パンはとてもおいしいです。

**翻譯：** 剛烤好的麵包非常好吃。

**解析：** 「焼きたて」（剛烤好）＋「の」修飾名詞「パン」。「〜たて」表示剛完成的新鮮狀態。

### 第2題 ✓
A：いつもお世話になっております。
B：いいえ、（　　）。

1. あなたこそ
2. **こちらこそ** ← 正解
3. そちらこそ
4. どちらこそ

**我的答案：2**

**正解句：**
> いいえ、こちらこそ。

**翻譯：** 哪裡哪裡，我才要感謝您。

**解析：** 「こちらこそ」是固定用法，用於謙虛地回應對方的感謝或問候，表示「我才應該說這句話」。

### 第3題 ✓
熱がひどくて、（　　）食べられない。

1. ご飯だけ
2. ご飯しか
3. **ご飯さえ** ← 正解
4. ご飯ばかり

**我的答案：3**

**正解句：**
> 熱[ねつ]がひどくて、ご飯[はん]さえ 食[た]べられない。

**翻譯：** 發高燒，連飯都吃不下。

**解析：** 「さえ」強調極端情況：連最基本的飯都吃不下，更別說其他東西了。

**選項比較：**
- だけ：只有～
- しか：只有～（後接否定）
- さえ：連～都～（強調極端）
- ばかり：淨是、光是～

### 第4題 ❌
日本へ（　　）ころは、日本語が全然わからなかった。

1. **来たての** ← 正解
2. 来るの
3. 来ての
4. 来たばかり

**我的答案：3**

**正解句：**
> 日本[にほん]へ 来[き]たての ころは、日本語[にほんご]が 全然[ぜんぜん]わからなかった。

**翻譯：** 剛來日本的時候，日語完全不懂。

**解析：**
- 「来**たて**」= 剛來（動詞連用形＋たて）
- 「来**て**の」這個形態**不存在**！
- 「来る」是カ變動詞，連用形是「来（き）」
- 来（き）+ たて = 来たて

**「〜たて」的接續：**
- 動詞連用形 ＋ たて
- 焼く → 焼き → 焼きたて
- 来る → 来（き）→ 来たて
- 揚げる → 揚げ → 揚げたて

### 第5題 ✓
毎日の努力（　　）成功の秘訣だ。

1. だけ
2. しか
3. **こそ** ← 正解
4. さえ

**我的答案：3**

**正解句：**
> 毎日[まいにち]の 努力[どりょく]こそ 成功[せいこう]の 秘訣[ひけつ]だ。

**翻譯：** 每天的努力才是成功的秘訣。

**解析：** 「こそ」強調「正是～才是～」，用於表達強烈的確信或核心論點。

---

## 重點整理

| 文法 | 意思 | 重點 |
|:---|:---|:---|
| 動詞連用形＋**たて** | 剛～的 | 焼きたて、来たて、揚げたて |
| 名詞＋**こそ** | 正是～、才是～ | 強調核心、固定用法「こちらこそ」 |
| 名詞＋**さえ** | 連～都～ | 強調極端情況，常接否定 |

## 錯題筆記

### 「〜たて」的形成
動詞連用形 ＋ たて（不是て形！）

| 動詞 | 連用形 | ＋たて |
|:---|:---|:---|
| 焼く | 焼き | 焼きたて |
| 来る | 来（き） | 来たて |
| 揚げる | 揚げ | 揚げたて |
| 炊く | 炊き | 炊きたて |
| 採る | 採り | 採りたて |
