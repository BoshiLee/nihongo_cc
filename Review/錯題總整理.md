# N3 錯題總整理

**最後更新：2026-01-27**

---

## 一、複合動詞 (2026-01-12)

| 文法 | 意思 | 關鍵判斷 |
| ---- | ---- | ---- |
| 〜合う | 互相做 | 看到「と」→ 想到「合う」 |
| 〜上がる | 完成（自動詞） | 「が」→ 上がる |
| 〜上げる | 完成（他動詞） | 「を」→ 上げる |
| 〜忘れる | 忘記做 | 不是所有動詞都能接「上げる」 |
| 〜たて | 剛做完 | 連用形＋たて（不是語幹！） |

**例句：**

- {部長|ぶちょう}と{話|はな}し<b><u>合う</u></b>（互相討論）
- {論文|ろんぶん}を{書|か}き<b><u>上げた</u></b>（寫完了）
- {家|いえ}が{建|た}ち<b><u>上がった</u></b>（蓋好了）
- {卒業|そつぎょう}<b><u>したて</u></b>（剛畢業）

## 二、對照與對應 (2026-01-14)

| 文法 | 意思 | 後接內容 |
| ---- | ---- | ---- |
| について | 關於（主題） | 討論、調查、說明 |
| に対して | 對於（對象） | 態度、行動、反應 |
| によって | 依據（基準） | 因…而異 |
| にとって | 對…而言（立場） | 主觀評價、形容詞 |
| として | 作為（身分） | 動作（働く、参加する） |

**判斷口訣：**

- 「說」什麼 → について
- 「對」什麼有態度 → に対して
- 「依」什麼不同 → によって
- 「對我來說」→ にとって
- 「以身分」做 → として

## 三、程度與強調 (2026-01-19)

| 文法 | 意思 | 關鍵詞 |
| ---- | ---- | ---- |
| たびに | 每當…就 | 重複動作、每次 |
| うちに | 趁著… | 機會、狀態持續 |
| こそ | 正是、才是 | 強調（今度こそ） |
| さえ…ば | 只要…就 | 唯一條件 |
| さえ…ない | 連…都不 | 極端例子 |
| ことにした | 自己決定 | 有意志 |
| ようになった | 變得… | 能力/習慣變化 |

**常見搭配：**

- {今度|こんど}<b><u>こそ</u></b>{成功|せいこう}したい（這次才要成功）
- {時間|じかん}<b><u>さえ</u></b>あれば{行|い}ける（只要有時間就能去）
- {挨拶|あいさつ}<b><u>さえ</u></b>しない（連打招呼都不做）

## 四、意圖與否定意志 (2026-01-21)

| 文法 | 意思 | 使用情境 |
| ---- | ---- | ---- |
| 〜ようとしない | 頑固不肯做 | 第三人稱抗拒 |
| つもりはない | 沒有打算 | 主觀否定計畫 |
| 〜まい（推測） | 應該不會吧 | 否定推測 |
| 〜まい（意志） | 絕不再… | 搭配「心に決めた」 |

**詞彙辨析：**

- {会社|かいしゃ}を<b><u>{継|つ}ぐ</u></b>（繼承公司）
- {会社|かいしゃ}に<b><u>{入|はい}る</u></b>（進入公司）

## 五、情感使役與被動 (2026-01-21 + 2026-01-26)

| 情況 | 用法 | 例 |
| ---- | ---- | ---- |
| 自己產生情緒 | 原形 | 私は泣いた |
| 讓別人產生情緒 | 使役形 | 彼女を泣かせた |
| 被引發情緒 | 使役被動 | 泣かせられた |

**常用情感使役：**

| 原形 | 使役形 | 意思 |
| ---- | ---- | ---- |
| 喜ぶ | 喜ばせる | 讓...高興 |
| 悲しむ | 悲しませる | 讓...傷心 |
| 怒る | 怒らせる | 讓...生氣 |
| 驚く | 驚かせる | 讓...驚訝 |
| 心配する | 心配させる | 讓...擔心 |

**主語判斷：**

- 主語是「事物」→ 使役形（ニュースがみんなを驚かせる）
- 主語是「人」→ 被動形（私は驚かされた）

## 六、気はない vs つもりはない (2026-01-26)

| 文法 | 意思 | 強調 |
| ---- | ---- | ---- |
| 気はない | 沒有意願 | 主觀態度（不想做） |
| つもりはない | 沒有打算 | 具體計畫 |

**例句：**

- どんなに{頼|たの}んでも、{手伝|てつだ}う<b><u>{気|き}はない</u></b>（不管怎麼拜託都沒意願幫忙）
- {大学|だいがく}を{卒業|そつぎょう}しても、{会社|かいしゃ}を{継|つ}ぐ<b><u>つもりはない</u></b>（畢業後也沒打算繼承公司）

## 總整理表

| 分類 | 弱點文法 | 記憶口訣 |
| ---- | ---- | ---- |
| 複合動詞 | 〜たて | 連用形＋たて |
| 對照對應 | について vs に対して | 說→について、態度→に対して |
| 程度強調 | さえ | 條件→さえ…ば、極端→さえ…ない |
| 否定意志 | 〜まい | 推測 or 決心 |
| 使役被動 | 情感使役 | 讓別人→使役形、被引發→使役被動 |
| 意願計畫 | 気はない vs つもりはない | 態度→気、計畫→つもり |
