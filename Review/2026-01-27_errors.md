# 複習清單 - 2026-01-27 錯題

練習前先複習這些題目！

## 錯題1｜たびに vs うちに

**題目：**
> この映画は見る（　　）涙が出る。

**正解：たびに**

> この{映画|えいが}は{見|み}る<b><u>たびに</u></b>{涙|なみだ}が{出|で}る。

**翻譯：** 這部電影每次看都會流淚。

**重點：**
- **たびに** = 每當…就（重複動作，每次都發生）
- **うちに** = 趁著…／在…過程中（狀態持續期間）
- 判斷關鍵：「每次」→ たびに、「趁著」→ うちに

## 錯題2｜さえ…ば vs こそ

**題目：**
> 薬（　　）飲めば、すぐ治りますよ。

**正解：さえ**

> {薬|くすり}<b><u>さえ</u></b>{飲|の}めば、すぐ{治|なお}りますよ。

**翻譯：** 只要吃藥，馬上就會好。

**重點：**
- **さえ…ば** = 只要…就（唯一條件）
- **こそ** = 強調，但不接條件形「〜ば」
- 看到「〜ば」→ 用「さえ」

## 錯題3｜〜ようとしない vs 気がしない

**題目：**
> 何度注意しても、彼は遅刻を（　　）。

**正解：やめようとしない**

> {何度|なんど}{注意|ちゅうい}しても、{彼|かれ}は{遅刻|ちこく}を<b><u>やめようとしない</u></b>。

**翻譯：** 不管提醒多少次，他就是不肯改掉遲到的習慣。

**重點：**
- **〜ようとしない** = 第三人稱頑固地拒絕做某事
- **気がしない** = 沒有那種感覺（用於直覺、預感）
- 判斷關鍵：描述他人抗拒 → ようとしない

## 錯題4｜〜まい（否定意志）

**題目：**
> あんな辛い経験は二度と（　　）と思う。

**正解：しまい**

> あんな{辛|つら}い{経験|けいけん}は{二度|にど}と<b><u>しまい</u></b>と{思|おも}う。

**翻譯：** 我想那樣痛苦的經歷絕不要再有了。

**重點：**
- **〜まい** = 否定意志（絕不再…）
- 「二度と」＋「〜まい」是固定搭配，表示強烈決心
- 「気はない」語氣較弱，不如「〜まい」強烈

## 快速記憶卡

| 文法 | 意思 | 關鍵點 |
| ---- | ---- | ---- |
| たびに | 每當…就 | 重複動作、每次 |
| うちに | 趁著… | 狀態持續期間 |
| さえ…ば | 只要…就 | 唯一條件（接「ば」） |
| こそ | 正是、才是 | 強調（不接「ば」） |
| 〜ようとしない | 頑固不肯做 | 第三人稱抗拒 |
| 〜まい | 絕不再… | 搭配「二度と」 |
