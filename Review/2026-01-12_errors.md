## 錯題1｜〜合う

**題目：**
> その問題について、部長と（　　）必要があります。

**正解：話し合う**

> その 問題[もんだい]について、部長[ぶちょう]と 話[はな]し合[あ]う 必要[ひつよう]があります。

**翻譯：** 關於那個問題，有必要和部長討論。

**重點：** 看到「**と**」→ 想到「**〜合う**」（互相）

---

## 錯題2｜〜上がる vs 〜上げる

**題目：**
> やっと論文を（　　）。本当に大変だった。

**正解：書き上げた**

> やっと 論文[ろんぶん]を 書[か]き上[あ]げた。本当[ほんとう]に 大変[たいへん]だった。

**翻譯：** 終於把論文寫完了。真的很辛苦。

**重點：**
```
を → 上げる（他動詞）
が → 上がる（自動詞）
```

## 錯題3｜〜忘れる

**題目：**
> 出かけるとき、エアコンを（　　）。

**正解：消し忘れた**

> 出[で]かけるとき、エアコンを 消[け]し忘[わす]れた。

**翻譯：** 出門的時候，忘記關冷氣了。

**重點：** 「消し**上げる**」不存在！「消す」只能接「〜忘れる」

## 錯題4｜〜たて

**題目：**
> 日本へ（　　）ころは、日本語が全然わからなかった。

**正解：来たての**

> 日本[にほん]へ 来[き]たての ころは、日本語[にほんご]が 全然[ぜんぜん]わからなかった。

**翻譯：** 剛來日本的時候，日語完全不懂。

**重點：** 「〜たて」接**連用形**，不是て形！
```
来る → 来（き）→ 来たて ✓
来る → 来て → 来ての ✗
```

## 錯題5｜自動詞 vs 他動詞（建つ/建てる）

**題目：**
> やっと家が（　　）。来月引っ越しできる。

**正解：建ち上がった**

> やっと 家[いえ]が 建[た]ち上[あ]がった。来月[らいげつ] 引[ひ]っ越[こ]しできる。

**翻譯：** 房子終於蓋好了。下個月可以搬家了。

**重點：** 自動詞配「が」，他動詞配「を」
```
家が建ち上がる（自動詞）✓
家を建て上げる（他動詞）
```

## 錯題6｜〜たて（卒業したて）

**題目：**
> 彼は大学を（　　）で、まだ社会経験が少ない。

**正解：卒業したて**

> 彼[かれ]は 大学[だいがく]を 卒業[そつぎょう]したてで、まだ 社会経験[しゃかいけいけん]が 少[すく]ない。

**翻譯：** 他剛大學畢業，社會經驗還很少。

**重點：** 「〜たて」接**連用形**，不是語幹！
```
卒業する → 卒業し（連用形）→ 卒業したて ✓
卒業する → 卒業（語幹）→ 卒業たて ✗
```

## 錯題7｜て形連接（目が合って）

**題目：**
> 友達と目が（　　）、二人とも笑ってしまった。

**正解：合って**

> 友達[ともだち]と 目[め]が 合[あ]って、二人[ふたり]とも 笑[わら]ってしまった。

**翻譯：** 和朋友眼神對上了，兩個人都笑了。

**重點：** て形用來連接兩個動作
```
動詞て形 + 動詞 = 表示動作順序
合って（て形）→ 可以連接後面的動作 ✓
合った（過去式）→ 句子結束 ✗
```

## 快速記憶卡

| 文法 | 口訣 |
|:---|:---|
| 〜合う | 「と」→「合う」 |
| 上がる/上げる | 「を」→「上げる」、「が」→「上がる」 |
| 〜忘れる | 不是所有動詞都能接「上げる」 |
| 〜たて | 連用形＋たて（不是語幹！） |
| 自他動詞 | 「が」→自動詞、「を」→他動詞 |
| て形連接 | 連接兩個動作表示順序 |
