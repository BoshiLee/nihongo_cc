## 錯題1｜〜ようとしない vs 〜まい

**題目：**
> 彼女は何を言っても、自分の意見を（　　）。

**正解：変えようとしない**

> {彼女|かのじょ}は{何|なに}を{言|い}っても、{自分|じぶん}の{意見|いけん}を<b><u>{変|か}えようとしない</u></b>。

**翻譯：** 不管說什麼，她就是不肯改變自己的意見。

**重點：**
- **〜ようとしない** = 第三人稱頑固地拒絕做某事
- **〜まい** = 第一人稱的否定意志（我絕不…）
- 判斷關鍵：主語是誰？第三人稱 → 〜ようとしない

## 錯題2｜うちに vs たびに

**題目：**
> 忙しい（　　）旅行に行っておきたい。

**正解：うちに**

> {忙|いそが}しい<b><u>うちに</u></b>{旅行|りょこう}に{行|い}っておきたい。

**翻譯：** 想趁還不忙的時候先去旅行。

**重點：**
- **うちに** = 趁著…（狀態持續期間）
- **たびに** = 每當…就（重複動作）
- 判斷關鍵：「趁著」→ うちに、「每次」→ たびに
- **〜ておく** = 事先做好（為將來準備）

## 錯題3｜迷惑受身

**題目：**
> 電車の中で足を（　　）、本当に迷惑だった。

**正解：踏まれて**

> {電車|でんしゃ}の{中|なか}で{足|あし}を<b><u>{踏|ふ}まれて</u></b>、{本当|ほんとう}に{迷惑|めいわく}だった。

**翻譯：** 在電車裡被踩到腳，真的很困擾。

**重點：**
- **迷惑受身** = 被別人做了某事而感到困擾（用被動形）
- 句型：「（身體部位）を＋被動形」
- ❌「踏ませて」= 讓人踩（使役）→ 不合邏輯

**踏む的動詞變化：**

| 形式 | 變化 | 意思 |
| ---- | ---- | ---- |
| 基本形 | {踏|ふ}む | 踩 |
| 被動形 | {踏|ふ}まれる | 被踩 |
| 使役形 | {踏|ふ}ませる | 讓人踩 |
| 使役被動形 | {踏|ふ}まされる | 被迫踩 |
