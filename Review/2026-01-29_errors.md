## 錯題1｜に対して vs によって

**題目：**
> 子供の頃は元気だった祖母（　　）、今は体が弱くなってしまった。

**正解：に対して**

> {子供|こども}の{頃|ころ}は{元気|げんき}だった{祖母|そぼ}<b><u>に{対|たい}して</u></b>、{今|いま}は{体|からだ}が{弱|よわ}くなってしまった。

**翻譯：** 相對於小時候很有精神的祖母，現在身體變弱了。

**重點：**
- **に対して** = 對比兩者的差異（A に対して B）
- **によって** = 根據、由於、因人而異
- 判斷關鍵：是「對比」還是「原因/手段」？

**比較表：**

| 文法 | 意思 | 例句 |
| ---- | ---- | ---- |
| に対して | 相對於～（對比） | 兄に対して、弟は静かだ |
| によって | 根據～／由於～ | 人によって意見が違う |
| について | 關於～（主題） | 日本について調べる |

## 錯題2｜するまい vs しまい

**題目：**
> 二度とあんな失敗は（　　）と心に決めた。

**正解：するまい**

> {二度|にど}とあんな{失敗|しっぱい}は<b><u>するまい</u></b>と{心|こころ}に{決|き}めた。

**翻譯：** 我下定決心絕不再犯那樣的失敗。

**重點：**
- **〜まい** = 第一人稱的否定意志「絕不再～」
- 「する」的〜まい形有兩種：

| 形式 | 說明 | 使用場合 |
| ---- | ---- | ---- |
| するまい | 標準形 | 書面語、正式場合 |
| しまい | 口語縮約形 | 日常對話 |

- JLPT 考試通常選 **するまい**（標準形）
