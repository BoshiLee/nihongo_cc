| 文法組合 | 問題點 | 解決方法 |
| ---- | ---- | ---- |
| 〜ようとしない vs 〜まい | 混淆第三人稱和第一人稱 | 第三人稱 → 〜ようとしない、第一人稱 → 〜まい |
| うちに vs たびに | 混淆「趁著」和「每次」 | 狀態期間 → うちに、重複動作 → たびに |
