| 文法組合 | 問題點 | 解決方法 |
| ---- | ---- | ---- |
| 〜ようとしない vs つもりはない | 混淆「抗拒態度」和「否定意圖」 | 第三人稱抗拒行為 → 〜ようとしない |
| 情感使役 | 誤用形容詞變化 | 讓某人產生情緒 → 使用「動詞使役形」 |
| 〜まい | 混淆推測與意志 | 否定推測：わかるまい、否定意志：しまい |
| 会社を継ぐ vs 入る | 詞彙選擇不夠道地 | 繼承家業使用「継ぐ」 |
