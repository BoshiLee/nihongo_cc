## 需要加強的文法

| 文法組合 | 問題點 | 解決方法 |
| ---- | ---- | ---- |
| について vs に対して | 混淆「主題」和「態度」 | 看後面動詞：說明→について、態度→に対して |
| にとって vs に対して | 混淆「評價」和「態度」 | 看後面詞性：形容詞→にとって、動作→に対して |
| として vs にとって | 混淆「身分」和「立場」 | として＋動作、にとって＋評價 |
| ことにした vs ようになった | 混淆「決定」和「變化」 | 自己決定→ことにした、能力變化→ようになった |
| さえ…ば vs さえ…ない | さえ 有多種用法 | 條件句→さえ…ば、極端例子→さえ…ない |
| こそ vs さえ | 混淆強調用法 | 強調→こそ、連…都→さえ |
