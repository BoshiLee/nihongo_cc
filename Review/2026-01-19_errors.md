## 錯題1｜たびに vs うちに

**題目：**
> 彼女に会う（　　）好きになっていく。

**正解：たびに**

> {彼女|かのじょ}に {会|あ}う<b><u>たびに</u></b> {好|す}きになっていく。

**翻譯：** 每次見到她就越來越喜歡。

**重點：**
```
たびに = 每當…就（重複動作，每次都發生）
うちに = 趁著… / 在…過程中（狀態持續期間）
```

**判斷方法：**
- 「每次」做某事 → **たびに**
- 「趁著」某狀態 → **うちに**

## 錯題2｜こそ（強調）

**題目：**
> 今度（　　）絶対に成功したい。

**正解：こそ**

> {今度|こんど}<b><u>こそ</u></b> {絶対|ぜったい}に {成功|せいこう}したい。

**翻譯：** 這次才一定要成功。

**重點：**
- 「今度こそ」是固定用法
- 表示「這次（跟之前不同）一定要…」

**常見搭配：**
| 表達 | 意義 |
|---|---|
| 今度こそ | 這次才要 |
| これこそ | 這才是 |
| だからこそ | 正因為如此 |

## 錯題3｜さえ…ば（唯一條件）

**題目：**
> 時間（　　）あれば、旅行に行きたい。

**正解：さえ**

> {時間|じかん}<b><u>さえ</u></b>あれば、{旅行|りょこう}に {行|い}きたい。

**翻譯：** 只要有時間，就想去旅行。

**重點：**
- 「さえ…ば」= 只要…就（唯一條件）
- 「こそ」是強調，不接條件形

**句型：**
```
名詞 + さえ + 條件形（あれば/いれば/すれば）
= 只要有…就
```
