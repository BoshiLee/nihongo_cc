## 五大易混淆文法總整理

```
┌─────────────┬──────────────┬─────────────┬────────────────────┐
│   文法       │    意思      │   後面常接   │       例句          │
├─────────────┼──────────────┼─────────────┼────────────────────┤
│  について    │ 關於（主題） │ 討論、調查   │ 日本について調べる  │
│  に対して    │ 對於（對象） │ 態度、行動   │ 先生に対して失礼だ  │
│  によって    │ 依據（基準） │ 違う        │ 国によって違う      │
│  として      │ 作為（身分） │ 動作        │ 医者として働く      │
│  にとって    │ 對…而言     │ 形容詞/評價  │ 私にとって大切だ    │
└─────────────┴──────────────┴─────────────┴────────────────────┘
```

## 判斷秘訣

### 1️⃣ について vs に対して

| 看後面動詞 | 選擇 |
|:---|:---|
| 説明する、話す、調べる、書く | **について** |
| 対応する、反対する、賛成する | **に対して** |

**口訣：**
- 「說」什麼 → **について**（主題）
- 「對」什麼有意見 → **に対して**（態度）

### 2️⃣ にとって vs に対して

| 看後面接什麼 | 選擇 |
|:---|:---|
| 形容詞（大切、難しい、大変） | **にとって** |
| 動作/態度（反対する、失礼だ） | **に対して** |

**口訣：**
- 「對我**而言**很重要」→ **にとって**
- 「對他**表示**反對」→ **に対して**

### 3️⃣ として vs にとって

| 看句子結構 | 選擇 |
|:---|:---|
| [身分] + 動作（工作、參加） | **として** |
| [人] + 評價（重要、難しい） | **にとって** |

**口訣：**
- 「**以**什麼身分做事」→ **として**
- 「**對**某人而言如何」→ **にとって**

### 4️⃣ によって

**固定搭配記憶：**
- ～によって**違う** = 因…而異
- ～によって**異なる** = 因…而不同

## 常錯陷阱

| 陷阱 | 錯誤 | 正確 |
|:---|:---|:---|
| 「對客訴應對」 | ~~クレームについて対応~~ | クレーム**に対して**対応 |
| 「對計畫贊成」 | ~~計画について賛成~~ | 計画**に対して**賛成 |
| 「對小孩而言難」 | ~~子供に対して難しい~~ | 子供**にとって**難しい |
| 「以口譯身分參加」 | ~~通訳にとって参加~~ | 通訳**として**参加 |

---

## 一句話總結

```
について = 關於什麼「主題」（討論、調查）
に対して = 對什麼「對象」（態度、行動）
によって = 因什麼「不同」（違う）
として  = 以什麼「身分」（動作）
にとって = 對誰「而言」（評價）
```
