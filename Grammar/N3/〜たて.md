# 〜たて

#N3文法 #複合動詞

## 基本意義
表示「剛〜的」，強調剛完成的新鮮狀態。動詞連用形 + たて。

---

## 用法

**接續：** 動詞連用形 + たて

**重要：** 接連用形，不是て形！

**例句：**
- **焼きたての**パンはとてもおいしいです。
  - 剛烤好的麵包非常好吃。
- 日本へ**来たての**ころは、日本語が全然わからなかった。
  - 剛來日本的時候，日語完全不懂。
- 彼は大学を**卒業したて**で、まだ社会経験が少ない。
  - 他剛大學畢業，社會經驗還很少。

---

## 接續變化

| 動詞 | 連用形 | + たて |
|:---|:---|:---|
| 焼く | 焼き | 焼きたて |
| 来る | 来（き） | 来たて |
| 揚げる | 揚げ | 揚げたて |
| 炊く | 炊き | 炊きたて |
| 採る | 採り | 採りたて |
| 卒業する | 卒業し | 卒業したて |

---

## 常見搭配

| 表達 | 意義 |
|:---|:---|
| 焼きたて | 剛烤好的 |
| 揚げたて | 剛炸好的 |
| 炊きたて | 剛煮好的（飯） |
| 採りたて | 剛採的 |
| 来たて | 剛來的 |
| 卒業したて | 剛畢業的 |

---

## 常見錯誤

| 正確 | 錯誤 |
|:---|:---|
| 焼き**たて** | 焼いて**の** |
| 来**たて** | 来て**の** |
| 卒業し**たて** | 卒業**たて** |

**記憶點：** 「〜たて」接連用形，不是て形，也不是語幹！
