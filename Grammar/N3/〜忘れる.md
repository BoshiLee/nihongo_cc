# 〜忘れる

#N3文法 #複合動詞

## 基本意義
表示「忘記做某動作」。動詞連用形 + 忘れる。

---

## 用法

**接續：** 動詞連用形 + 忘れる

**例句：**
- 今朝、薬を**飲み忘れて**しまった。
  - 今天早上不小心忘了吃藥。
- 出かけるとき、エアコンを**消し忘れた**。
  - 出門的時候，忘記關冷氣了。
- 会議の資料を**準備し忘れて**、上司に怒られた。
  - 忘記準備會議資料，被上司罵了。

---

## 常見搭配

| 複合動詞 | 意義 |
|:---|:---|
| 飲み忘れる | 忘記吃（藥） |
| 消し忘れる | 忘記關 |
| 書き忘れる | 忘記寫 |
| 置き忘れる | 忘記帶走 |
| 言い忘れる | 忘記說 |
| 閉め忘れる | 忘記關（門窗） |
| 準備し忘れる | 忘記準備 |

---

## 常見句型

「〜忘れる」常搭配：
- **〜てしまった**：表示遺憾
  - 薬を飲み忘れて**しまった**（不小心忘了吃藥）
- **〜たまま**：表示維持狀態
  - 窓を閉め忘れた**まま**出かけた（忘記關窗就出門了）
