# 〜合う

#N3文法 #複合動詞

## 基本意義
表示「互相做某動作」。動詞連用形 + 合う。

---

## 用法

**接續：** 動詞連用形 + 合う

**例句：**
- 二人は長い時間**話し合って**、やっと意見がまとまった。
  - 兩人討論了很長時間，意見終於統一了。
- あの二人は学生時代からずっと**愛し合っている**。
  - 那兩人從學生時代就一直互相相愛。
- チームで働くときは、お互いに**助け合う**ことが大切だ。
  - 團隊工作時，互相幫助很重要。

---

## 常見搭配

| 複合動詞 | 意義 |
|:---|:---|
| 話し合う | 互相討論 |
| 助け合う | 互相幫助 |
| 愛し合う | 互相相愛 |
| 見つめ合う | 互相凝視 |
| 相談し合う | 互相商量 |
| 教え合う | 互相教導 |

---

## 重點

- 常搭配「**と**」表示共同對象
- 常搭配「**お互いに**」強調雙向性
- 「目が合う」是固定用法，表示「眼神對上」
