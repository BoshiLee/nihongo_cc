# 〜かえる

#N3文法 #複合動詞

## 基本意義
表示「重新做」「換〜」。動詞連用形 + かえる。

---

## 用法

**接續：** 動詞連用形 + かえる

**例句：**
- 東京駅で新幹線から在来線に**乗りかえて**ください。
  - 請在東京站從新幹線換乘在來線。
- この文章は間違いが多いから、**書きかえて**ください。
  - 這篇文章錯誤很多，請重寫。

---

## 常見搭配

| 複合動詞 | 意義 |
|:---|:---|
| 乗りかえる | 換乘 |
| 書きかえる | 改寫、重寫 |
| 入れかえる | 替換、更換 |
| 着がえる | 換衣服 |
| 言いかえる | 換句話說 |
| 考えかえる | 重新考慮 |

---

## 注意事項

「乗りかえる」是下一段動詞：
- て形：乗りかえ**て** ✓
- 乗りかえ**って** ✗（錯誤形態）
