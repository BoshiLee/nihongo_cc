# 情感使役

#N3文法 #使役與被動

---

## 變化規則

情感動詞 → **使役形**（讓某人產生該情緒）

| 類型 | 原形 | 使役形 | 意思 |
| ---- | ---- | ---- | ---- |
| 五段 | 喜ぶ | 喜ばせる | 讓...高興 |
| 五段 | 驚く | 驚かせる | 讓...驚訝 |
| 五段 | 怒る | 怒らせる | 讓...生氣 |
| 五段 | 泣く | 泣かせる | 讓...哭 |
| 五段 | 笑う | 笑わせる | 讓...笑 |
| 五段 | 困る | 困らせる | 讓...困擾 |
| 五段 | 悩む | 悩ませる | 讓...煩惱 |
| 一段 | 悲しむ | 悲しませる | 讓...傷心 |
| 一段 | 苦しむ | 苦しませる | 讓...痛苦 |
| 不規則 | 心配する | 心配させる | 讓...擔心 |
| 不規則 | 安心する | 安心させる | 讓...安心 |
| 不規則 | がっかりする | がっかりさせる | 讓...失望 |

---

## 例句

### 1. 喜ばせる（讓...高興）

> {母|はは}を<b><u>{喜|よろこ}ばせたくて</u></b>、{誕生日|たんじょうび}にケーキを{作|つく}った。

想讓媽媽高興，所以生日時做了蛋糕。

---

### 2. 悲しませる（讓...傷心）

> {嘘|うそ}をついて、{彼女|かのじょ}を<b><u>{悲|かな}しませて</u></b>しまった。

因為說謊，讓她傷心了。

---

### 3. 怒らせる（讓...生氣）

> また{遅刻|ちこく}して、{先生|せんせい}を<b><u>{怒|おこ}らせた</u></b>。

又遲到了，惹老師生氣。

---

### 4. 心配させる（讓...擔心）

> {連絡|れんらく}しなくて、{両親|りょうしん}を<b><u>{心配|しんぱい}させて</u></b>ごめん。

抱歉沒聯絡，讓父母擔心了。

---

### 5. 驚かせる（讓...驚訝）

> この{結果|けっか}は{皆|みんな}を<b><u>{驚|おどろ}かせる</u></b>だろう。

這個結果應該會讓大家嚇一跳吧。

---

### 6. 笑わせる（讓...笑）

> {彼|かれ}はいつも{面白|おもしろ}い{話|はなし}で{皆|みんな}を<b><u>{笑|わら}わせる</u></b>。

他總是用有趣的故事逗大家笑。

---

### 7. がっかりさせる（讓...失望）

> {期待|きたい}に{応|こた}えられなくて、<b><u>がっかりさせて</u></b>すみません。

沒能達到期待，讓你失望了，抱歉。

## 核心概念：誰讓誰產生情緒？

```
┌─────────────────────────────────────────────────┐
│                    情感使役                       │
├─────────────────────────────────────────────────┤
│                                                  │
│   【原因/引發者】  →  讓  →  【感受者】產生情緒    │
│      (主語)                    (を)              │
│                                                  │
│   ニュース が    →  讓  →   私たち を 悲しませた  │
│   (新聞)                      (我們)   (傷心)     │
│                                                  │
└─────────────────────────────────────────────────┘
```

---

## 三種情況對比

| 情況 | 句型 | 例句 | 翻譯 |
| ---- | ---- | ---- | ---- |
| **自己感到** | 主語＋が＋情感動詞 | 私**が**悲しん**だ** | 我傷心了 |
| **讓別人感到** | 主語＋が＋對象＋を＋使役形 | ニュース**が**私たち**を**悲しま**せた** | 新聞讓我們傷心 |
| **被引發情緒** | 主語＋が＋使役被動 | 私**が**悲しま**せられた** | 我被弄得很傷心 |

---

## 判斷方法

**Step 1：找主語是誰？**

- 主語是「感受情緒的人」→ 用**原形**或**被動**
- 主語是「引發情緒的事物/人」→ 用**使役形**

**Step 2：看助詞**

- 「〜**を**＋情感動詞」→ 幾乎一定是**使役形**
- 因為情感動詞本身是自動詞，不接「を」

---

## 實例分析

**例1：** 子供たち**を**喜ば**せる**ために、手品を見せた。

- 「子供たちを」= 對象（被讓高興的人）
- 主語省略（我）讓孩子們高興 → **使役形**

**例2：** ニュース**が**私たち**を**悲しま**せた**。

- 主語「ニュース」= 引發情緒的事物
- 「私たちを」= 感受者
- 新聞讓我們傷心 → **使役形**

**例3：** 私**は**悲しん**だ**。

- 主語「私」= 感受者自己
- 自己傷心 → **原形**

---

## 常見錯誤

| 錯誤 | 正確 | 原因 |
| ---- | ---- | ---- |
| ニュースが私たちを悲し**んだ** | 悲しま**せた** | 主語是事物，要用使役 |
| 子供を喜**ぶ** | 喜**ばせる** | 讓別人高興，要用使役 |
| 私を悲しま**せた** | 私**が**悲し**んだ** | 自己傷心，用原形 |

---

## 關鍵判斷（簡表）

| 情況 | 用法 | 例 |
| ---- | ---- | ---- |
| 自己產生情緒 | 原形 | 私は悲しんだ（我傷心了） |
| 讓別人產生情緒 | 使役形 | 彼女を悲しませた（讓她傷心） |
| 被引發情緒 | 使役被動 | 悲しませられた（被弄得傷心） |
